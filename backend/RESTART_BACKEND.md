# Как перезапустить бэкенд

## Что такое терминал бэкенда?

**Терминал бэкенда** - это окно PowerShell (черное или синее окно), где запущена команда:
```
python -m uvicorn main:app --reload
```

## Как найти терминал бэкенда?

1. Посмотрите на **панель задач Windows** (внизу экрана)
2. Найдите иконку **PowerShell** или окно с текстом
3. Откройте его - там должны быть логи типа:
   ```
   INFO:     Uvicorn running on http://127.0.0.1:8000
   INFO:     Application startup complete
   ```

Если не видите такое окно - бэкенд может быть запущен в фоне или не запущен.

## Как перезапустить бэкенд?

### Способ 1: Если видите окно терминала

1. Найдите окно терминала с бэкендом
2. Нажмите **Ctrl+C** для остановки
3. Запустите снова:
   ```powershell
   cd C:\Users\User\Documents\PCHMI\backend
   .venv\Scripts\Activate.ps1
   python -m uvicorn main:app --reload
   ```

### Способ 2: Если не видите окно (запуск заново)

1. Откройте **новое окно PowerShell**
2. Выполните команды:
   ```powershell
   cd C:\Users\User\Documents\PCHMI\backend
   .venv\Scripts\Activate.ps1
   python -m uvicorn main:app --reload
   ```

## Как проверить, что бэкенд работает?

1. Откройте в браузере: **http://localhost:8000/**
   - Должно отобразиться: `"Onlearn API"`

2. Откройте: **http://localhost:8000/docs**
   - Должен открыться Swagger UI с документацией API

## Что делать, если не работает?

1. Проверьте, что бэкенд запущен (см. выше)
2. Проверьте логи в терминале бэкенда - там будут видны ошибки
3. Убедитесь, что порт 8000 не занят другим приложением
