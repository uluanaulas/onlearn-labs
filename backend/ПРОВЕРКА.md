# Как проверить, что аутентификация работает

## Шаг 1: Убедитесь, что бэкенд запущен

В окне PowerShell должно быть:
```
INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete
```

## Шаг 2: Попробуйте войти в браузере

1. Откройте: `http://localhost:8080`
2. Войдите: `alice@example.com` / `password123`
3. Попробуйте записаться на курс

## Шаг 3: Смотрите логи в окне PowerShell

При каждом запросе должны появляться логи:
```
[AUTH] Token received: eyJhbGciOiJIUzI1NiIsInR5cCI6Ik...
[AUTH] Decoded user_id: 1
[AUTH] User authenticated: Alice Johnson
```

## Если логов [AUTH] нет:

1. Бэкенд не перезапустился с новым кодом
2. Нужно перезапустить бэкенд:
   - В окне PowerShell нажмите `Ctrl+C`
   - Запустите снова: `python -m uvicorn main:app --reload`

## Если видите ошибки:

Скопируйте текст из окна PowerShell и отправьте мне - там будет видно, в чем проблема.
